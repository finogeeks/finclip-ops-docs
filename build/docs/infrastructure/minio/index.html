minio

 FinClip还用到对象存储，用于存储小程序包、图片等静态资源文件。对象存储是业界非常成熟的存储方案，在客 户的实际部署中，我们支持使用第三方对象存储，例如腾讯云COS、阿里云OSS、AWS S3等等。除此之外，在私 有化部署中，我们可以使用Minio进行部署对象存储服务服务器。
